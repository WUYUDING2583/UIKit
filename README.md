### Add the JitPack repository to your build file

```css
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

### Add the dependency

```css
dependencies {
	        implementation 'com.github.yuyi2583:UIKit:Tag'
	}
```
